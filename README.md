## ✨ Funcionalidades Principais

### 🤖 Automação Visual (Flow Builder)
Crie fluxos complexos de atendimento sem escrever uma única linha de código. Use o nosso construtor drag-and-drop para definir caminhos, condições de validação, atrasos e disparar mídias (áudio, vídeo, imagem).

### 🧠 Agentes de IA Inteligentes
Integração nativa com modelos GPT e Claude para que sua IA entenda o contexto das conversas, responda dúvidas frequentes e qualifique leads de forma autônoma 24/7.

### 📊 Dashboard Estratégico
Visão completa da saúde do seu negócio. Filtros avançados por período e mês, métricas de novos leads, conversão de funil e volume de mensagens em tempo real.

### 📂 Funil de Vendas (Pipeline)
Gerencie sua jornada de vendas através de estágios configuráveis. Arraste leads entre as colunas, atribua tags automaticamente e acompanhe cada oportunidade de perto.

### 🚀 Disparo em Massa (Batch Dispatch)
Envie milhares de mensagens via planilha (CSV/XLSX) de forma segura. Nosso algoritmo utiliza delays inteligentes (5-15s) para garantir o máximo de entrega e o mínimo de risco de banimento.

## 🛠 Stack Tecnológica

### Frontend
- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Estilização**: Tailwind CSS + Shadcn UI
- **Fluxos Visuais**: React Flow
- **Estado**: Zustand
- **Real-time**: WebSocket (Socket.io)

### Backend
- **Core**: Node.js + Express + TypeScript
- **Banco de Dados**: PostgreSQL (via Prisma ORM)
- **Integração WhatsApp**: Evolution API
- **Infraestrutura**: Docker & Docker Compose
